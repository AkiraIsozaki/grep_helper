================================================================
wheelhouse/ ディレクトリについて（Solaris 10 SPARC オフライン用）
================================================================

このディレクトリは Solaris 10 SPARC のオフライン環境で依存パッケージを
インストールするための wheel ファイルを格納します。

================================================================
■ wheel ファイルの準備（インターネット接続環境で実行）
================================================================

【方法1: make を使う】

    make download-wheels

【方法2: pip を直接使う】

    pip download -r requirements.txt -d wheelhouse

どちらも Linux / Mac の開発環境で実行してください。
生成された wheelhouse/ ディレクトリごと Solaris 環境に持ち込みます。

================================================================
■ 対象パッケージ
================================================================

    javalang（純粋Pythonパッケージ）

javalang は C拡張を持たない純粋Pythonパッケージです。
wheel ファイルのタグは "py3-none-any" であり、OS・CPUアーキテクチャを
問わず使用できます。Linux で取得した whl を SPARC でもそのまま使えます。

================================================================
■ Solaris 10 SPARC での使い方
================================================================

1. Linux/Mac 環境で wheel を取得:
       make download-wheels

2. 本ツール一式（wheelhouse/ を含む）を Solaris 10 へ転送

3. Solaris 10 で Python 3.7 以上を準備
   （ホストOSに Python 3 がない場合は別途持ち込む）

4. セットアップ実行:
       ./setup_solaris.sh

   Python のパスを明示する場合:
       PYTHON_CMD=/opt/python37/bin/python3.7 ./setup_solaris.sh

5. 実行:
       ./run_solaris.sh --source-dir /path/to/javaproject
